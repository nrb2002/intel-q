# Implementation Guide – UI Components

## Step 1

Install shadcn/ui

```bash
npx shadcn@latest init
```

---

## Step 2

Generate components

```bash
npx shadcn add button
```

Repeat for

- Card
- Input
- Dialog
- Badge

---

## Step 3

Create custom components

```
LoadingSpinner.tsx

Toast.tsx

EmptyState.tsx

QueueStatusBadge.tsx
```
